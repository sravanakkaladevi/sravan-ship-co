# 🚢 Sravan Ship Co — Ship Manufacture & Luxury Cruises

A premium, interactive 3D scroll-animated landing page showcasing state-of-the-art naval engineering, ship manufacturing, and luxury cruises. 

Created, designed, and developed solely by **Sravan**.

---

## 🌟 Key Features

* **3D Scroll Canvas Animations**: Fluid, frame-by-frame scroll-driven rendering of naval vessels and luxury cruises powered by HTML5 Canvas.
* **High-DPI Screen Support**: Support for high-resolution displays (Retina/4K screens) using dynamic `devicePixelRatio` scaling.
* **Smooth Page Navigation**: Smooth scroll links from the header to individual page sections (Home, Voyages, Experience, About, Contact).
* **Fully Responsive**: Optimized for desktop, tablet, and mobile layouts with fluid sizing (`clamp()`).
* **Luxury Branding**: Complete branding, customized typography, premium color themes, and modern bento-grid layouts.
* **Pricing in INR**: Curated packages and suite upgrades shown in Indian Rupees (₹).

---

## 🛠️ Tech Stack

* **Core Framework**: [Next.js](https://nextjs.org/) (v16.2.9) & [React](https://react.dev/) (v19)
* **Logic & Types**: [TypeScript](https://www.typescriptlang.org/)
* **Animations**: [GSAP (GreenSock)](https://gsap.com/) & [ScrollTrigger](https://gsap.com/docs/v3/Plugins/ScrollTrigger/)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/) & Vanilla CSS variables
* **Icons & UI**: [Playwright](https://playwright.dev/) for testing, custom icon indicators

## 🚀 Getting Started

Follow these steps to run the application locally:

### 1. Install Dependencies
```bash
npm install
```

### 2. Start the Development Server
```bash
npm run dev
```

### 3. Open the App
Open your browser and navigate to:
* **Local**: `http://localhost:3000`
* **Network**: `http://<your-ip-address>:3000`
